专业 技能

前端的学习需要持之以恒，知识深度与广度都很重要。而前端开发的重中之重就是原生javascript的理解，而如今的思路必需条理清晰。

HTML

html是超文本标记语言,通过使用标记来描述文档结构和表现形式的一种语言,由浏览器进行解析,然后把结果显示在网页上,它是网页构成的基础.

CSS

css是层叠样式表，用于控制网页样式，是能够真正做到网页表现与内容分离的一种样式设计语言.

Java Script

js是面向对象，基于事件和对象驱动的解释性、松散型语言.为浏览器的一部分，广泛用于客户端的脚本语言，用来给HTML网页增加动态功能.

JQUERY

JQuery是继prototype之后又一个优秀的Javascript库。它是轻量级的js库 ，它兼容CSS3，还兼容各种浏览器,核心理念是write less,do more(写得更少,做得更多).

HTML5

htnl5万维网的核心语言、标准通用标记语言下的一个应用超文本标记语言（HTML）的第五次重大修改.

CSS3

CSS即层叠样式表（Cascading StyleSheet）。 在网页制作时采用层叠样式表技术，可以有效地对页面的布局、字体、颜色、背景和其它效果实现更加精确的控制.
